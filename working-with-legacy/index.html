Le code hérité

D'où vient le legacy ?

Du développeur : Manque d'expérience

  • Faible compétence en design
  • Faire "Quand on aura le temps"
  • Pas de refactoring
  • Manque de temps pour essayer / apprendre

De la team : manque de communication

  • Personne ne dit non
  • Faible connaissance métier
  • Faible connaissance technique
  • Fonctionnalités trop complexe / inadaptés
  • Bricolage de l'existant

De l'organisation : rigide

  • Urgence constante
  • Turnover
  • Peu de communication avec le métier
  • Pas d'investissement technique
